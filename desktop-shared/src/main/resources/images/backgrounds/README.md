# Background Images

Place the following JPEG files here (recommended: 1920×1080, compressed ~200–400 KB each):

| File | Theme |
|------|-------|
| `bg_forest_mist.jpg` | Forest mist — works with Sage, Nord |
| `bg_ocean_horizon.jpg` | Ocean horizon — works with Ocean, Nord |
| `bg_cherry_blossom.jpg` | Cherry blossoms — works with Rose, Sepia |
| `bg_mountain_snow.jpg` | Mountain snow — works with Nord, Indigo |
| `bg_city_night.jpg` | City at night — works with Dark, Indigo |
| `bg_parchment.jpg` | Paper/parchment texture — works with Sepia, Light |

These are bundled resources — users select from this preset list in Appearance Settings.
A semi-transparent overlay (theme surface color at ~0.55 opacity) is applied on top 
so UI text remains readable regardless of background choice.

All current images gotten from https://unsplash.com/, they are commercial free.
